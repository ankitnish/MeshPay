# MeshPay

A secure offline digital payment platform built with the MERN stack.

## 🚀 Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Socket.IO

### Security
- JWT Authentication
- AES-256 Encryption
- RSA Encryption
